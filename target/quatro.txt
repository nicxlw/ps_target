Enunciado - Descubra a lógica e complete o próximo elemento:
a) 1, 3, 5, 7, ___
b) 2, 4, 8, 16, 32, 64, ____
c) 0, 1, 4, 9, 16, 25, 36, ____
d) 4, 16, 36, 64, ____
e) 1, 1, 2, 3, 5, 8, ____
f) 2,10, 12, 16, 17, 18, 19, ____

Respostas:

a) Sequência de números ímpares. O próximo elemento é 9.
b) Sequência de potências de 2. O próximo elemento é 128.
c) Sequência onde cada elemento exceto o primeiro é a soma do elemento anterior com o próximo número ímpar da sequência 1, 3, 5 (...) O elemento 36 é a soma de 25 com o número ímpar 11, logo, o próximo elemento é 36 + 13 = 49.
d) Sequência é composta por quadrados de números pares consecutivos. Como 64 = 8^2, o próximo elemento é 10^2 = 100
e) Sequência de Fibonacci. O próximo elemento é 13.
f) Essa sequência não segue uma lógica matemática, mas de grafia. Lembro de ter visto essa sequência em uma aula de matemática do ensino médio. Os números começam com a letra D. O próximo elemento é 200.
